# 🚀 Rajesh Kumar Behera - Futuristic Developer Portfolio

A futuristic and modern developer portfolio built with React, TypeScript, Tailwind CSS, and Framer Motion featuring premium UI/UX, smooth animations, responsive layouts, AI-focused branding, and interactive sections.

---

# 🌟 Features

- Modern Futuristic UI Design
- Fully Responsive Layout
- Smooth Framer Motion Animations
- Interactive Skills Showcase
- Dynamic Projects Section
- Certifications & Achievements
- Working Contact Form using Web3Forms
- Resume Download Feature
- Social Media Integration
- AI + Cyber Security Focused Portfolio

---
